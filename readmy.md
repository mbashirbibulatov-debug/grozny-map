# Интерактивная карта Грозного
Мой первый проект на Python.

### Что умеет этот код:
* Генерирует интерактивную карту города.
* Использует циклы `for` для автоматической расстановки меток.
* Работает со структурами данных (списки и словари).
* Выводит информацию об объектах в красивых всплывающих окнах.

**Стек:** Python 3, Folium.
